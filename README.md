# webgl-svg-demo

Демонстрационный проект, показывающий совместную работу WebGL и SVG-графики на одной веб-странице.

На странице размещены два отдельных контейнера. В первом с помощью библиотеки Three.js отображаются два куба, во втором - SVG-изображение круга. При нажатии на кнопку кубы сближаются, а после завершения анимации оба 3D-объекта и SVG-круг синхронно меняют цвет, при повторном нажатии кнопки - объекты возвращаются в исходное состояние. Проект демонстрирует взаимодействие 3D-сцены и 2D-векторной графики в рамках единого визуального сценария.

## Возможности

- Отображение двух кубов в диметрической проекции
- Использование ортографической камеры
- Использвоание амбиентного и направленного света
- Анимация сближения и отдаления объектов по нажатию кнопки
- Синхронное изменение цвета объектов WebGL и SVG
- Модульная структура проекта

## Технологии

- Three.js
- SVG
- JavaScript (ES6 Modules)
